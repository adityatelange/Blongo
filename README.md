# Blongo 
Blongo is an open-source Blog Web-App built on the Django Web Framework 2.x with Python3

## About Blongo

#### Features
* Generation of Atom and RSS feeds
* Generation of Sitemaps

#### Dependencies
* Django
* pytz
* whitenoise
